üìã Integration Instructions

Add this code to your website before the closing </body> tag: 

<script>
  window.RagChatConfig = {
  apiKey: "your-api-key-here",           // Required: Your RAG Chat API key
  // author: "custom-user-id",           // Optional: Visitor identifier (auto-generated UUID if not provided)
  position: "bottom-right",              // Optional: 'bottom-right' or 'bottom-left'
  primaryColor: "#007bff",               // Optional: Brand color for the chat button
  baseUrl: "https://yourapp.com"         // Optional: Your domain (defaults to current)
};
</script>
<script
  src="https://yourapp.com/widget/loader.js"
  async
></script>

Note: Replace your-api-key-here with your actual API key and https://yourapp.com with your domain.
